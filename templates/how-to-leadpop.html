Create header

Based on the site's design, create a 515px x 113px header image, must include the background and dealer logo only. You can get the assets from the actual site by using the google chrome inspector tool. If you cross paths with a site that has a slideshow as a background, take the picture you like the most to build the header.

Name the image file according to the format: leadpopup_shortname.jpg

Upload the picture to the live server
