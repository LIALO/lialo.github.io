Glossary

  • Carpeta del cliente

    Carpeta en el servidor (Stage o Live) que contiene los archivos del sitio. Usualmente estará nombrada según el nombre corto del cliente.

  • Config

    Es el archivo donde están las variables de configuración del cliente, lo puedes encontrar en la ruta: /includes dentro de la carpeta del cliente.

  • Content PSDs

    Es la carpeta dentro del servidor fred que almacena los PSD que se generan durante algunas tareas de content, está ubicada en la ruta: /Common.

  • Domain Name

    The Live URL for a certain client, which might be the one that the client is currently using, or the one he will use when the stage site go Live. In the doneness doc, you will find the domain name in the row 5 for every customer column.

  • Doneness Doc

    It's a Google Spreadsheet that contains the information of the clients. There are two of this documents, one for Redesign (most of them) and another one for New Clients.

  • Fred

    Server where the PSD files are saved, among many other files. In order to conect to this server, you must be connected to the VPN.
